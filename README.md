# land use and land cover change

## Installation

Install pipenv 
```
$ pip install pipenv
```
Change to the project directory

```
$ cd project_name
```
Install the required libraries 

```
$ pipenv install 
```
Activate your pipenv environment
```
$ pipenv shell 
```
